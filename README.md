# Aspro.Cloud AI Ассистент

Веб-чат с ИИ-ботом, подключённым к вашей CRM Аспро.Cloud через MCP.  
Разворачивается на **Vercel** за 10 минут, без сервера, без домена, без разработчиков.

## Как это работает

```
Сотрудник (браузер) → Vercel (чат + API) → OpenRouter (ИИ-модель)
                                          → Aspro.Cloud (данные CRM)
```

Сотрудник открывает сайт, задаёт вопрос → ИИ-модель получает данные из CRM через MCP → отвечает на вопрос.

## Пошаговая инструкция развёртывания на Vercel

### Шаг 1. Подготовьте ключи

Вам нужны три вещи:

| Что | Где взять |
|-----|-----------|
| **OpenRouter API-ключ** | [openrouter.ai/keys](https://openrouter.ai/keys) — зарегистрируйтесь и создайте ключ |
| **Код портала Aspro.Cloud** | Из URL вашего портала: `mycompany.aspro.cloud` → `mycompany` |
| **API-ключ Aspro.Cloud** | Настройки портала → Настройки API → Добавить |

### Шаг 2. Загрузите проект на GitHub

1. Зайдите на [github.com](https://github.com) (создайте аккаунт, если нет).
2. Нажмите **«+»** → **«New repository»**.
3. Назовите, например, `aspro-chat`, сделайте **Private**.
4. Загрузите в него все файлы этого проекта (можно через кнопку «Upload files» на GitHub).

### Шаг 3. Разверните на Vercel

1. Зайдите на [vercel.com](https://vercel.com) и авторизуйтесь через GitHub.
2. Нажмите **«Add New…» → «Project»**.
3. Выберите ваш репозиторий `aspro-chat` → нажмите **«Import»**.
4. Vercel автоматически определит, что это Next.js проект.
5. **Раскройте «Environment Variables»** и добавьте переменные:

   | Name | Value |
   |------|-------|
   | `OPENROUTER_API_KEY` | `sk-or-v1-ваш_ключ` |
   | `ASPRO_ACCOUNT_CODE` | `mycompany` |
   | `ASPRO_API_KEY` | `ваш_api_ключ_aspro` |
   | `ASPRO_MCP_URL` | `https://mcp.aspro.cloud/mcp` |
   | `AI_MODEL` | `anthropic/claude-sonnet-4` |

6. Нажмите **«Deploy»**.
7. Через 1–2 минуты сайт будет доступен по адресу типа `aspro-chat.vercel.app`.

### Шаг 4. Готово!

Откройте ссылку — вы увидите чат. Отправьте ваш первый вопрос, например:
> «Сколько открытых сделок на портале?»

Этой ссылкой можно поделиться с сотрудниками — у всех будет доступ.

## Выбор модели

По умолчанию используется `anthropic/claude-sonnet-4`. Вы можете сменить модель, изменив переменную `AI_MODEL` в настройках Vercel. Популярные варианты:

| Модель | Переменная AI_MODEL | Комментарий |
|--------|---------------------|-------------|
| Claude Sonnet 4 | `anthropic/claude-sonnet-4` | Отличный баланс цены и качества |
| Claude Haiku 4.5 | `anthropic/claude-haiku-4-5` | Быстрый и дешёвый |
| GPT-4o | `openai/gpt-4o` | Альтернатива от OpenAI |
| Gemini 2.5 Flash | `google/gemini-2.5-flash-preview` | Быстрый, дешёвый |

После смены переменной нажмите **Redeploy** в Vercel.

## Безопасность

⚠️ **Важно**: по умолчанию чат доступен всем, кто знает ссылку.

Для ограничения доступа можно:

1. **Vercel Authentication** (платная функция Vercel Pro) — ограничивает доступ по email.
2. **Добавить пароль**: установите переменную `CHAT_PASSWORD` и добавьте проверку на фронтенде (потребуется небольшая доработка кода).
3. **Vercel Firewall**: ограничьте доступ по IP.
